SenangWebs One - Auto Initialization

This instance of SWO is initialized automatically because the div below has the data-swo attribute. It will use the default code or load previously saved code from localStorage (if data-swo-storage-key is set or a default one is used).

The SWO library's JavaScript (dist/swo.js) includes logic to find all elements with data-swo on DOMContentLoaded and initialize them.
